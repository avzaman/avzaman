Ultrasound Semi-Supervised Segmentation

  • Python
  • Pytorch
  • AI/ML
  • Benchmarking/Evaluaion
  • Diffusion-Model
  • Data-Pipelining
  • CUDA-API
  • GPU-Cluster-Programming

A research project conducted to enhance the performance of fully supervised image segmentation by using fake generated images. The datasets are breast tumor ultrasounds. First I measure benchmarks on the fully supervised learning technique using Deeplab-V3 pretrained as a base. Next I train a latent embedder to reduce the labeled images into a latent space to then train a diffusion model. Next, using the diffusion model I generate 1000 fake tumor ultrasounds. Lastly I use the labeled images and the generated images to conduct mean-teacher training (a semi-supervised method) and check if metrics have improved. The metrics did not improve substantially.

US Utility Rates Disparity Analysis

  • MatplotLib
  • SQL
  • Research

As a research project I used datamining techniques to attempt to divulge unique patterns that show up relating to the US Census demographic data in 2020 to the Utility Prices report from the US Energy Administration. Using python libraries including pandas, geopandas, numpy, matlotlib, and sklearn I clean and join the datasets on different geographical conditions (zipcode and state).
